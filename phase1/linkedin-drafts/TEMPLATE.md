# LinkedIn Post Template

**Rules:**
- Maximum 150 words total
- No motivational content — technical observations only
- Always link to the actual GitHub work, not just the repository root
- Do not post if you have nothing real to share

---

**Hook (line 1):** A specific technical thing you discovered or found surprising.

**Body (3–4 sentences):** What the concept is. Why it matters. One real-world connection — embedded device, real CVE, real incident.

**Close:** Full analysis/code in my GitHub repository: [link]
