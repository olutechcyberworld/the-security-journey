## What this repository is

This is not a course completion tracker. It is a public record of deliberate, evidence-based security engineering work across 18 months minimum. Every commit represents something built, analysed, exploited, or explained — not something watched or read.

The work here spans offensive security foundations, binary exploitation, reverse engineering, cryptography, embedded systems security, and firmware analysis. It is organized by phase and week, with supporting CVE analyses, Python tooling, and technical write-ups.

## The Rule That Does Not Change

Motion is not progress. Watching tutorials, reading articles, and attending classes are preparation at best. The only thing that counts is a working result — code that runs, a challenge solved, a write-up pushed, a concept you can explain without notes.
